# ESP.Utilites.GetCenter
```lua
<<Vector3> Position> ESP.Utilites.GetCenter(<Instance> Object)
```
This returns the center position of the `Object`.

## Parameters
* `Vector3` Object
> The object you want to get the center of

## Returns
* `Vector3` Position
